# 💰 Financial Calculator (Java)

A simple financial calculator written in Java that supports:
- 💵 Money operations (`+`, `-`)
- 💱 Currency conversion (`USD`, `EUR`)
- 🧠 Nested expressions and functions
- 🧩 Parsing expressions from strings
